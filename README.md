# PDC (Parallel and Distributed Computing)

This repository contains programs and tasks from the Parallel and Distributed Computing course.

---

## Repository Structure

### Chapter 1/
- Task 1/
  - `task1_process.py` → Python code using multiprocessing  
  - `task1_thread.py` → Python code using threading  
  - `task1_process_output.png` → Output & execution time for process code  
  - `task1_thread_output.png` → Output & execution time for thread code  

### Chapter 2/
- `lock` → Example demonstrating Lock concept  
- `rlock` → Example demonstrating RLock (Reentrant Lock)  
- `semaphore` → Example demonstrating Semaphore
